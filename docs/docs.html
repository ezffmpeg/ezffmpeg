Getting Started

Create and edit videos with JavaScript using ezffmpeg.

Install with npm

npm install ezffmpeg

Basic usage

const ezffmpeg = require('ezffmpeg');

const project = new ezffmpeg({
  width: 1080,
  height: 1920
});

async function createVideo() {
  await project.load([
    {
      type: 'video',
      url: 'input.mp4'
    }
  ]);
  
  await project.export({
    outputPath: 'output.mp4'
  });
}

createVideo();

Constructor

Create a new ezffmpeg instance with your desired settings.

const project = new ezffmpeg({
  width: 1920,    // Output video width
  height: 1080,   // Output video height
  fps: 30,        // Frames per second (optional)
  audioCodec: 'aac', // Audio codec (optional)
  videoCodec: 'libx264' // Video codec (optional)
});

Available options

Option Type Default Description
width number 1920 Output video width in pixels
height number 1080 Output video height in pixels
fps number 30 Frames per second
audioCodec string 'aac' Audio codec to use
videoCodec string 'libx264' Video codec to use

load() API

Load media elements (videos, images, audio, text) into your project.

await project.load([
  {
    type: 'video',
    url: 'video.mp4',
    position: 0,
    end: 10,
    cutFrom: 2,
    cutTo: 8
  },
  {
    type: 'audio',
    url: 'audio.mp3',
    position: 0,
    end: 15
  },
  {
    type: 'text',
    text: 'Hello World',
    position: 5,
    end: 10,
    fontSize: 48,
    fontColor: 'white'
  }
]);

Media options

Video options

Option Type Description
type string Must be 'video'
url string Path or URL to the video file
position number Start position in seconds in the output video
end number End position in seconds in the output video
cutFrom number Optional: Start cutting from this position in the source video
cutTo number Optional: End cutting at this position in the source video
rotation number Optional: Rotation angle in degrees

Audio options

Option Type Description
type string Must be 'audio'
url string Path or URL to the audio file
position number Start position in seconds in the output video
end number End position in seconds in the output video
volume number Optional: Volume level (1.0 is normal)

Text options

Option Type Description
type string Must be 'text'
text string The text to display
position number Start position in seconds in the output video
end number End position in seconds in the output video
fontSize number Font size in pixels
fontColor string Font color (CSS color name or hex)
x number Optional: X position (default: centered)
y number Optional: Y position (default: centered)

export() API

Export your project to a video file.

await project.export({
  outputPath: 'output.mp4',
  quality: 'high',
  overwrite: true
});

Export options

Option Type Default Description
outputPath string required Path where the output video will be saved
quality string 'medium' Video quality: 'low', 'medium', 'high'
overwrite boolean false Whether to overwrite existing files

Examples

Concatenate multiple videos

await project.load([
  { type: 'video', url: 'video1.mp4', position: 0 },
  { type: 'video', url: 'video2.mp4', position: 5 },
  { type: 'video', url: 'video3.mp4', position: 10 }
]);

await project.export({ outputPath: 'concatenated.mp4' });

Add text overlay

await project.load([
  { type: 'video', url: 'background.mp4', position: 0 },
  { 
    type: 'text', 
    text: 'Hello World', 
    position: 2, 
    end: 8,
    fontSize: 64,
    fontColor: 'white'
  }
]);

await project.export({ outputPath: 'text_overlay.mp4' });
